<HighD_Dataset>
OriginalRaw Datasets from "https://levelxdata.com/highd-dataset/"


<Dataset_after_Processing>
Enhance data with searching and aligning surrouding vehicles' informations for ego vehicle. Prepare the data for the training of velocity predicition(H_HTP) or trajectory prediction(VectorNet_HighD).
Generated by "H_HTP/data_processing/highd_dp.py"


<PyG_DataSet>
The reading speed on .csv file is too slow, it is not friendly for the repearting training and testing of predictor. so we transfer the format of data saving from .csv to .pyg.
Based on the <Dataset_after_Processing>, it is generated by "H_HTP/data_processing/HighD_datapre_pyg_velocity.py" or "VectorNet_HighD/HighD_datapre_pyg.py"
